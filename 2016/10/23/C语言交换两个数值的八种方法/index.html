简单总结，以鞭笞岁月。


编译器:XCode8.0

异或

异或一般是针对二进制数.

  1. 两数异或，相同则得0，不同则得1。
  2. x ^ x = 0，x ^ 0 = x
    第二行: m:0101 n:0011
    第三行: m:0110 n:0011
    第四行: n:0101 m:0110
    第五行: m:0011 n:0101
    此时，已完成交换.

运算符

  1. 从左向右依次进行，首先获取m与n的和，接着利用()运算符直接将m的值赋予n，此时n完成交换。
  2. 同时()内的值是m的值，两数之和减去m值便是n的值，再将n的值赋予m，到这里m完成交换。

位运算

注:__int64表示64位整型变量，Xcode中用long代替，VC6.0可直接使用__int64，即: b = (__int64)((__int64)a << 32 | (a = b)) >> 32；

  1. (m = n):完成m的交换，()内的值是n。
  2. (long)((long)m << 32 | n):得到的结果是 00...0 0101 00...0 11第一部分28个0，第三部分30个0 。
  3. ((long)((long)m << 32 ) >> 32:得到的结果:把第三第四部分去掉，即00...0 0101 正好32位，且此值为m的值，再将其赋予n，完成n的交换。

知识

int最大值，根据编译器类型不同而变化。

  1. 对于16位编译器，int占16位(2字节)。int的最大值为32767。
  2. 对于32位和64位编译器，int占32位(4字节)。int的最大值为2147483647。
  3. 可以通过打印sizeof(int)查看平台对应的int占用字节数。乘8后即为位数。最高位为符号位，如位数为n，则最大值为2^(n-1) 即2的n-1次幂。
